# laravel-resourses-demo
Demonstration of api application based on laravel resourses

####1
git clone repo
####2
cp .env.example .env
####3
composer install
####4
php artisan key:generate
####5
touch database/database.sqlite
####6
php artisan migrate
####7
php artisan jwt:secret
####8
See routes: php artisan route:list
####9
Have some fun!
